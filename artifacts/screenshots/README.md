Screenshots placeholder and instructions

This folder should contain three images used for reviewer evidence:

- `upload_page.png` — screenshot of the upload page with files attached
- `claims_list.png` — screenshot of the claims list showing the created claim
- `claim_details.png` — screenshot of the claim details page showing decision and reasons

Current session produced screenshots available in the interactive session. To finalize the artifacts, copy the saved images from your browser session or take fresh screenshots and place them here with the exact filenames above.

If you need the exact images captured in this session, they can be retrieved from the running browser session resources or re-generated by re-running the following steps locally:

1. Start backend and frontend
2. Run `python scripts/verify_endpoints.py` to create a claim and upload sample files
3. Open `http://localhost:3000/upload`, `http://localhost:3000/claims`, and the claim detail URL and take screenshots
